Procura e imprime todos os caminhos entre duas antenas.

Utiliza pesquisa em profundidade para encontrar e imprimir todos os caminhos possíveis entre duas antenas distintas, mostrando as suas coordenadas.

Parameters
cidadeApontador para a cidade onde será feita a pesquisa.
inicioÍndice da antena de partida.
destinoÍndice da antena de chegada.
caminhoArray onde será guardado o caminho atual.
tamanhoComprimento atual do caminho (normalmente começa a 0).
Returns
int Retorna 1 (imprimiu) se pelo menos um caminho for encontrado.
int Retorna 0 (imprimiu) se não houver caminho.
int Retorna -1 se não existir a cidade.
int Retorna -2 se não for válida a antena início.
int Retorna -3 se não for válida a antena destino.
int Retorna -4 se não for válida a antena início e destino.
int Retorna -5 se não houver antenas.
